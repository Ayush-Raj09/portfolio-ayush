# Ayush Raj - Data Analyst Portfolio

Welcome to the source code for my professional portfolio website.

## Overview
This monolithic, high-performance HTML/CSS/JS portfolio serves as a showcase of my skills, projects, and educational background as a Data Analyst. It was custom-built with precision and speed in mind, featuring:
